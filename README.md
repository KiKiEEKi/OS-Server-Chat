# OS-Server-Chat
Бот для общего чата между сервером и дискордом.

Требования:
- Metamod
- CounterStrikeSharp
- DSharpPlus

Переменные:
- Файл настроек <Config.json> в папке плагина
```json
{
    "Token": "token", //Токен дискорд бота
	"Channel": 13129575645 //Айди канала куда и откуда будут отправляться сообщения
}
```
Установка:
- Закинуть папку OSServerChat в папку ../csgo/addons/counterstrikesharp/plugins/
